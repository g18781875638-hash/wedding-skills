# 婚礼客户需求采集问卷生成器

## 简介
这是一个 Claude Code 的 Skill，用于快速生成结构化的婚礼客户需求采集问卷。它模拟资深婚礼策划师兼主持人的思维，分 6 步引导新人梳理需求，最终输出一份可直接发给客户确认的【婚礼需求简报】。

## 适用场景
- 初次接触新人客户，需要快速了解其需求
- 标准化团队的客户需求采集流程
- 避免因沟通不充分导致后期修改和纠纷

## 使用方法
1. 确保你的 Claude Code 已加载本 Skill（项目目录下启动或全局安装均可）
2. 在 Claude Code 对话中输入：
   > 用婚礼客户需求采集问卷生成器，帮一位（城市）的（婚礼类型）新人做一份需求问卷
3. Claude Code 会分 6 步引导你完成：
   - 基本信息（姓名、婚期、城市、宾客规模）
   - 爱情故事（相识相恋过程、最打动对方的瞬间）
   - 婚礼调性（草坪清新、室内奢华、小型私密等）
   - 流程偏好（First Look、才艺、感恩环节等）
   - 特殊环节（惊喜环节、忌讳）
   - 宾客互动（游戏、抽奖）
4. 完成后，Claude Code 会输出一份结构化的【婚礼需求简报】，可直接复制发给客户
